1.1.0
- เพิ่มชนิดไฟล์ *.hnk
- เพิ่มการแยกมัลติซาวด์ฟ้อนสำหรับชุดกลอง
- เพิ่มการบิวด์ด้วย MinGW สำหรับเครื่องที่มีปัญหากับ Visual C++ Redistributable 2015
- เพิ่มออปชั่นการ เปิด-ปิด การใช้ 32-bit floating-point และเสียงเอฟเฟ็กต์ สำหรับ Midi synthesizer
- ปรับให้การเลือกมัลติซาวด์ฟ้อนยังคงอยู่ หากมีการ เพิ่ม-ลบ รายการซาวด์ฟ้อน
- ปรับปรุงการทำงานภายในของ Midi synthesizer
- SongsHelper
    - เพิ่มส่วนของการแปลง NCN เป็น HNK
    - เพิ่มส่วนของการเพิ่มแปลง HNK  

1.0.0
- รองรับชื่อโฟล์เดอร์ และชื่อไฟล์ภาษาไทย
- แก้การวาดเนื้อเพลง ให้วาดได้เร็วขึ้น
- แก้ EQ ความถี่ต่ำกว่า 80 Hz
- แก้ปัญหาการปรับปรุงฐานข้อมูลช้า
- เพิ่มการปรับความเร็วเพลง ( Page Up, Page Down )
- เพิ่มคีย์ลัด Esc เพื่อซ่อนการค้นหา รายการเพลง และรายละเอียดเพลง
- เพิ่มคีย์ลัด Ctrl+x เพื่อลบข้อความภายในช่องค้นหา
- เพิ่มตัวเลือกย่อขนาดเนื้อเพลงอัตโนมัติ เมื่อล้นขอบจอด้านซ้ายขวา
- เพิ่มตัวเลือกตำแหน่งเนื้อเพลง กึ่งกลาง ชิดซ้าย และ ชิดขวา
- เพิ่มตัวเลือกสีและขนาดเส้นขอบนอกของเนื้อเพลง

- เปลี่ยน ไอคอนและโทนสีของโปรแกรมเป็นสีฟ้า
